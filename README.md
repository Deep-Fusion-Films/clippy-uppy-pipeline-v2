# Clippy Uppy Pipeline v2
### AI‑powered, event‑driven video ingestion & enrichment pipeline

Clippy Uppy Pipeline v2 is a **modular, serverless system** for ingesting, processing, and enriching video content using **Google Cloud** and **Gemini models**.  
It supports multiple providers, real‑time and batch processing, and produces structured metadata stored in Firestore and GCS.
